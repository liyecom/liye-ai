# T3: Phase Transition - Stockout Cascade

## Pattern Type
**相变模式**：系统状态的突然切换，从一个稳定状态跳跃到另一个完全不同的状态。

## Description
断货不是"少卖一点"，而是**系统性崩塌**：
1. 库存归零 → Listing 排名暴跌
2. 排名暴跌 → 广告无法投放或效率极低
3. 补货后 → 需要从头积累动量

这是**不可逆的相变**：恢复成本远高于预防成本。

## Conditions
- 库存周转过快，安全库存不足
- 供应链有不确定性（节假日、物流延迟）
- 销售预测不准确

## Early Signals
- 库存天数 < 14 天
- 补货周期延长（物流拥堵信号）
- 销售速度突然加快（可能耗尽库存）
- 竞品开始断货（可能意味着供应链普遍紧张）

## What To Do
- 在库存 < 21 天时触发预警
- 宁可多压库存，也不能断货
- 分散仓库（FBA + FBM 备份）
- 销售加速时立即评估补货能力

## What This Pattern Doesn't Predict
- 断货多久后排名会恢复
- 竞品是否会趁机抢占你的位置
- 恢复后的销售曲线会是什么样

## Similar Phase Transitions
- 差评突增导致转化率断崖下跌
- 账户被封导致全部 Listing 下线
- 类目审查导致产品被移除
