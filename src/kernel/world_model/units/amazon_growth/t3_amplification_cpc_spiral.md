# T3: Amplification - CPC Spiral

## Pattern Type
**放大模式**：小的变化被市场机制放大，形成难以逆转的趋势。

## Description
当多个卖家同时提高竞价时，CPC 上涨会自我强化：
1. 你提高竞价 → 竞争对手跟进
2. 竞争对手跟进 → 你需要再次提高
3. 循环往复 → CPC 飙升

这不是线性增长，而是**非线性放大**。

## Conditions
- 类目中有 3+ 个主要卖家在同时竞争
- 关键词搜索量有限（供给固定）
- 卖家对广告 ROI 有不同的容忍度

## Early Signals
- CPC 涨幅 > 搜索量涨幅
- 广告位排名波动剧烈
- 小幅调整竞价就导致显著排名变化
- 竞品广告投放频率增加

## What To Do
- 避免在螺旋初期参与"军备竞赛"
- 设置 CPC 硬上限，到达后不再追加
- 寻找替代关键词分散风险
- 考虑暂时退出观望，等螺旋结束

## What This Pattern Doesn't Predict
- 螺旋什么时候会停止
- 谁会先退出
- 最终稳定的 CPC 是多少
