# T1: Budget Reflexivity Spiral

## Failure Mode
预算增加 → 竞价提高 → CPC 上涨 → 预算消耗更快 → 需要更多预算

这是一个**正反馈螺旋**：你的行为改变了市场价格，而市场价格又影响你的结果。

## Trigger Conditions
- 在竞争激烈的关键词上大幅增加预算
- 多个卖家同时提高竞价（节日季、促销期）
- 产品评分或库存状态突然变化导致转化率下降

## Early Signals
- CPC 连续 3 天上涨超过 10%
- 相同预算的点击数持续减少
- 竞争对手广告位排名变化频繁

## Stop Signals
- 当 CPC 涨幅超过 ROI 容忍阈值时，停止加预算
- 当竞价已达到盈亏平衡点时，不再提高出价
- 当发现多个竞争对手也在加价时，考虑暂停观望

## Key Assumptions (if this fails)
- 假设市场价格不会因为你的行为而变化（错误）
- 假设竞争对手不会跟进你的策略（错误）

## What This Doesn't Tell You
- 这不告诉你：竞争对手的预算上限是多少
- 这不告诉你：CPC 上涨是暂时的还是新常态
- 这不告诉你：是否有更便宜的替代关键词
