# 📐 TES Model 完整技术文档

**TES**: Traffic Efficiency Score（流量效能分数）
**Version**: v2.0
**最后更新**: 2025-12-21

---

## 一、核心公式

```
TES = (月搜索量 × 购买率) / (标题密度 + 1)
```

**英文表述**:
```
TES = (Monthly Search Volume × Purchase Rate) / (Title Density + 1)
```

**符号说明**:
- `TES`: Traffic Efficiency Score（流量效能分数）
- `月搜索量`: Monthly Search Volume（估算月度搜索次数）
- `购买率`: Purchase Rate（关键词级别的转化率，单位：%）
- `标题密度`: Title Density（第一页有多少个产品的标题包含该关键词）
- `+1`: 平滑因子（避免除以0，并降低极端值影响）

---

## 二、公式组成部分详解

### 2.1 月搜索量 (Monthly Search Volume)

**定义**:
在 Amazon 美国站，该关键词在一个自然月内被用户搜索的预估次数。

**数据来源**:
SellersSprite（卖家精灵）基于大数据 AI 预测模型（2019年后准确率约90%）

**数值示例**:
- `door mat indoor entrance`: **83,661** 次/月
- `keep off rug`: **8,500** 次/月
- `tapetes para entrada de casa`（西班牙语）: **4,200** 次/月

**特殊情况处理**:
- 如果显示为 `0` 或 `NULL` → 表示搜索量低于索引阈值，不参与计算
- 如果是季节性关键词 → 非旺季显示为 `0`（如 "Halloween doormat" 在12月）
- 高搜索量词（Top 200）可能有 ±30% 误差

**数据展示逻辑**:
```
UI 上下两行：
上行：83,661 （月搜索量）
下行：2,789  （日均搜索量 = 83,661 / 30）
```

**ETL 字段名**:
- 原始字段: `月搜索量`
- 数据库字段: `search_volume` (INTEGER)
- 单位: 次/月

---

### 2.2 购买率 (Purchase Rate)

**定义**:
关键词级别的转化率 = 通过该关键词产生的购买量 / 该关键词的搜索量

**公式**:
```
购买率 = 月购买量 / 月搜索量
```

**数值示例**:
- `keep off rug`: **16.06%** （超高转化，精准需求）
- `entry rugs for inside house`: **7.88%** （高转化）
- `door mat indoor entrance`: **4.12%** （中等转化）
- `best door mat 2024`: **0.8%** （低转化，信息类搜索）

**战略阈值**:
- `购买率 > 10%`: **黄金词段** - 高精准度，用户决策速度快
- `购买率 5-10%`: **优质词** - 较高转化，值得投入
- `购买率 2-5%`: **一般词** - 中等转化，需优化
- `购买率 < 2%`: **弱词** - 低转化，可能是信息类/研究类搜索

**影响因素**:
1. **关键词精准度**: "washable door mat" > "door mat" （功能词 > 品类词）
2. **用户购买意图**: "buy door mat" > "door mat ideas" （购买词 > 研究词）
3. **产品-关键词匹配度**: 关键词越描述产品特性，转化率越高

**ETL 字段名**:
- 原始字段: `购买率` （可能显示为 "16.06%" 或 "0.1606"）
- 数据库字段: `purchase_rate` (DECIMAL)
- 单位: 百分比（存储为小数，0.1606 = 16.06%）

---

### 2.3 标题密度 (Title Density)

**定义**:
在该关键词搜索结果的第一页（通常48个产品），有多少个产品的标题中包含该关键词。

**数值示例**:
- `door mat`: **48** （几乎所有第一页产品都包含，红海市场）
- `door mat indoor entrance`: **38** （高密度，竞争激烈）
- `keep off rug`: **5** （低密度，蓝海机会）
- `tapetes para entrada de casa`: **2** （极低密度，竞争极小）

**战略意义**:
- **高密度 (>30)**:
  - ✅ 说明市场已高度优化
  - ❌ 难以靠 Listing 优化获得自然排名提升
  - 💡 策略：需要靠广告 + 销量突破

- **中密度 (10-30)**:
  - ⚖️ 有一定竞争但仍有机会
  - 💡 策略：优化 Listing + 中等竞价广告

- **低密度 (<10)**:
  - 🔥 蓝海机会
  - 💡 策略：快速优化 Listing 即可获得排名

**计算逻辑**:
```
假设搜索 "washable door mat"：
第一页 48 个产品中：
- 20 个标题包含 "washable door mat" → 标题密度 = 20
- 28 个标题不包含完整词组
```

**注意事项**:
- 标题密度 ≠ 竞争度（还需结合广告竞品数、ABA集中度）
- 标题密度 = 0 时，公式分母变为 1（平滑因子作用）

**ETL 字段名**:
- 原始字段: `标题密度`
- 数据库字段: `title_density` (INTEGER)
- 单位: 个（产品数量）

---

### 2.4 平滑因子 "+1" 的设计逻辑

**为什么要 +1？**

**原因 1: 避免除以零**
```
假设关键词 "tapetes de entrada"（西班牙语新词）
标题密度 = 0（第一页无人优化）

如果公式是: TES = (4,200 × 0.25) / 0  → 无穷大 ❌

使用 +1 后: TES = (4,200 × 0.25) / (0 + 1) = 1,050 ✅
```

**原因 2: 降低极端值影响**
```
对比两个关键词：
A. 标题密度 = 1  vs  B. 标题密度 = 2

不加 +1:
A: TES = 1000 / 1 = 1,000
B: TES = 1000 / 2 = 500
差距: 2 倍

加 +1 后:
A: TES = 1000 / 2 = 500
B: TES = 1000 / 3 = 333
差距: 1.5 倍（更平滑）
```

**原因 3: 符合实际竞争逻辑**
```
标题密度从 0 → 1 时:
- 即使只有 1 个竞品优化，也存在竞争
- +1 确保即使 density=0，分母仍 ≥1
```

## 三、TES 计算实例

### 案例 #1: 超高 TES 黄金词

**关键词**: `keep off rug`

**原始数据**:
- 月搜索量: 8,500
- 购买率: 16.06%（即 0.1606）
- 标题密度: 5

**计算过程**:
```
TES = (8,500 × 0.1606) / (5 + 1)
    = 1,365.1 / 6
    = 227.5
```

**评级**: 🔥🔥🔥 **超高效能词**

**战略建议**:
- ✅ 立即添加到标题/五点/后台
- ✅ 建立独立 Exact Match 广告组
- ✅ 竞价 $1.20-1.50（高于市场平均）
- ✅ Top of Search 位置溢价 +100%
- 🎯 目标: 7-14 天内推至自然排名 Top 10

---

### 案例 #2: 高流量但低效能词

**关键词**: `door mat`

**原始数据**:
- 月搜索量: 150,000（超高流量）
- 购买率: 3.2%（低转化）
- 标题密度: 48（红海）

**计算过程**:
```
TES = (150,000 × 0.032) / (48 + 1)
    = 4,800 / 49
    = 97.96
```

**评级**: ⚠️ **流量大但效能中等**

**战略建议**:
- ⚠️ 不适合精准匹配（竞争太激烈）
- ✅ 可用于 Broad Match 捡漏（低竞价 $0.50-0.80）
- ✅ 必须埋入标题（品类词必备）
- ❌ 不作为核心推广词（ACOS 会很高）

---

### 案例 #3: 西班牙语蓝海词

**关键词**: `tapetes para entrada de casa`

**原始数据**:
- 月搜索量: 4,200
- 购买率: 9.5%
- 标题密度: 2（极低竞争）

**计算过程**:
```
TES = (4,200 × 0.095) / (2 + 1)
    = 399 / 3
    = 133
```

**评级**: 🔥🔥 **高效能蓝海词**

**战略建议**:
- 🔥 黄金机会（高转化 + 低竞争）
- ✅ 立即建立西班牙语广告活动
- ✅ 竞价仅需 $0.40-0.60（远低于英文词）
- ✅ 预期 ACOS < 25%
- 💡 考虑添加西班牙语 A+ 内容

---

### 案例 #4: 信息类低价值词

**关键词**: `how to clean door mat`

**原始数据**:
- 月搜索量: 3,800
- 购买率: 0.5%（极低转化）
- 标题密度: 1

**计算过程**:
```
TES = (3,800 × 0.005) / (1 + 1)
    = 19 / 2
    = 9.5
```

**评级**: ❌ **低效能词**

**战略建议**:
- ❌ 添加为否定关键词（Negative Keyword）
- ❌ 不投放任何广告
- ❌ 不添加到 Listing
- 💡 用户是在找教程，不是买产品

## 四、TES 分层策略

基于 Timo 店铺实战经验的分层标准：

### 分层表格

| TES 区间 | 等级 | 战略定位 | 匹配类型 | 竞价策略 | 预算占比 |
|----------|------|----------|----------|----------|----------|
| **TES > 100** | 🔥 **赢家词 (Winner)** | 核心推广词 | Exact Match | 高竞价 ($1.20-2.00) | 50-60% |
| **TES 50-100** | ⭐ **潜力词 (Potential)** | 次核心词 | Exact + Phrase | 中竞价 ($0.80-1.20) | 25-30% |
| **TES 10-50** | 💎 **长尾词 (Longtail)** | 捡漏词 | Phrase + Broad | 低竞价 ($0.40-0.80) | 15-20% |
| **TES < 10** | ❌ **弱词 (Weak)** | 否定或不投 | - | 不投放 | 0% |

### 详细策略说明

#### 🔥 TES > 100: Winner 赢家词

**特征**:
- 高搜索量 × 高转化率 × 低竞争度
- "黄金三角"：流量、转化、机会的最佳平衡

**执行动作**:
```
Listing 优化:
- ✅ 必须出现在标题前 80 字符内
- ✅ 至少在 2 条五点描述中出现
- ✅ 添加到后台关键词（如果标题/五点未覆盖）

PPC 广告:
- Campaign: "SP-Exact-Winner"
- Match Type: Exact Match
- Bid: $1.20-2.00（视竞争度）
- Placement: Top of Search +100-150%
- Daily Budget: $30-50（单独给足预算）

预期效果:
- 7-14 天内推至自然排名 Top 20
- ACOS 目标: 25-35%
- ROI: 高（长期降低广告依赖）
```

**B08SWLTTSW 案例**:
- `keep off rug` (TES 227.5)
- `entry rugs for inside house` (TES 238.6)

---

#### ⭐ TES 50-100: Potential 潜力词

**特征**:
- 中高流量或中高转化
- 竞争适中

**执行动作**:
```
Listing 优化:
- ✅ 出现在标题或五点（至少覆盖1处）
- ⚠️ 优先级低于 Winner 词

PPC 广告:
- Campaign: "SP-Exact-Potential"
- Match Type: Exact + Phrase Match
- Bid: $0.80-1.20
- Placement: Product Pages +30%（关联流量）
- Daily Budget: $20-30

预期效果:
- 14-21 天内推至自然排名 Top 30
- ACOS 目标: 30-40%
```

**B08SWLTTSW 案例**:
- `washable door mat` (TES 79.5)
- `non slip door mat` (TES 78.5)

---

#### 💎 TES 10-50: Longtail 长尾词

**特征**:
- 低搜索量但可能高转化
- 或高搜索量但高竞争/低转化

**执行动作**:
```
Listing 优化:
- ⚠️ 选择性添加到五点/后台
- ❌ 不占用标题字符数

PPC 广告:
- Campaign: "SP-Broad-Discovery"
- Match Type: Phrase + Broad Match
- Bid: $0.40-0.80（低价捡漏）
- Placement: 不设置溢价
- Daily Budget: $10-20

预期效果:
- 捡漏策略，不强求排名
- ACOS 目标: 35-50%（可接受较高 ACOS）
- 主要目的：发现意外高转化词
```

**B08SWLTTSW 案例**:
- `mudroom rug washable` (TES 45.1)
- `outdoor front door mat` (TES 38.2)

---

#### ❌ TES < 10: Weak 弱词

**特征**:
- 信息类搜索（"how to", "best", "review"）
- 低转化率（<2%）
- 或配件类搜索（"door mat replacement", "mat only"）

**执行动作**:
```
Listing 优化:
- ❌ 不添加

PPC 广告:
- ❌ 添加为否定关键词（Negative Keyword）
- Campaign: 所有自动/宽泛广告活动
- Match Type: Phrase Negative

目的:
- 防止预算浪费
- 避免拉低整体转化率
```

**B08SWLTTSW 案例**:
- `how to clean door mat` (TES 9.5)
- `door mat ideas` (TES 4.2)
- `best door mat 2024` (TES 6.8)

## 五、TES 模型的优势与局限

### ✅ 优势

**1. 综合评估，避免片面决策**
```
传统做法：只看搜索量
- "door mat" 150,000 搜索量 → 投！
- 结果：ACOS 60%+，利润率为负

TES 方法：
- "door mat" TES 97.96 → 只做 Broad 捡漏
- "keep off rug" TES 227.5 → 核心推广
- 结果：ACOS 25-30%，利润率健康
```

**2. 发现蓝海机会**
```
低密度 + 高转化 = 黄金机会
- 传统方法看不到小搜索量词
- TES 会将这些词排到前列
```

**3. 可量化、可对比**
```
团队协作时统一标准：
- "这个词 TES 150，值得投"
- 避免主观判断（"我觉得这个词不错"）
```

**4. 动态调整**
```
TES 会随市场变化：
- 标题密度↑ → TES↓ → 降低投入
- 购买率↑ → TES↑ → 加大投入
```

---

### ⚠️ 局限性

**1. 忽略了品牌知名度**
```
"Nike shoes" 的购买率可能很低（浏览多，购买少）
但品牌词仍然重要（品牌保护）

解决方案：
- 品牌词单独管理，不用 TES 评估
```

**2. 忽略了季节性**
```
"Halloween door mat" 10月 TES 500，11月 TES 20

解决方案：
- 按月分析 TES
- 季节性词提前 1-2 月布局
```

**3. 依赖卖家精灵数据准确性**
```
如果卖家精灵数据不准（尤其搜索量）
TES 会失真

解决方案：
- 交叉验证（对比 Amazon 搜索建议频率排名）
- 以 TES 为参考，结合实际广告数据调整
```

**4. 不考虑 CPC 成本**
```
两个词 TES 相同，但 CPC 差 3 倍
实际 ROI 不同

解决方案：
- 引入 TES/CPC 比率（流量效能/成本比）
- TES/CPC > 100 = 超高性价比
```

## 六、TES 进阶应用

### 6.1 TES 动态监控

**每月计算 TES 变化**:
```
2025-11:
"keep off rug" TES = 227.5
  月搜索量: 8,500
  购买率: 16.06%
  标题密度: 5

2025-12:
"keep off rug" TES = 180.2 (下降 20.8%)
  月搜索量: 8,200 (下降 3.5%)
  购买率: 15.1% (下降 6.0%)
  标题密度: 7 (上升 40%) ← 主因

分析：
- 竞品开始优化这个词（标题密度增加）
- 需要加大广告投入以保持排名
```

---

### 6.2 TES/CPC 性价比分析

**公式**:
```
TES/CPC Ratio = TES / Suggested CPC

示例:
"keep off rug"
- TES = 227.5
- CPC = $1.20
- Ratio = 227.5 / 1.20 = 189.6

"door mat indoor entrance"
- TES = 95.3
- CPC = $1.80
- Ratio = 95.3 / 1.80 = 52.9

结论：
尽管 "door mat indoor entrance" 搜索量更大
但 "keep off rug" 的 性价比 高 3.6 倍
```

---

### 6.3 TES 区域化调整（西班牙语市场）

**问题**:
西班牙语关键词的标题密度通常极低（因为很少卖家优化）
这会导致 TES 虚高

**解决方案**:
```
西班牙语词 TES 计算时，调整平滑因子：

TES_ES = (月搜索量 × 购买率) / (标题密度 + 3)

原因：
- +3 而非 +1：因为西班牙语市场密度普遍低
- 避免所有西班牙语词 TES 都过高

示例:
"tapetes para entrada de casa"
- 原 TES = (4,200 × 0.095) / (2+1) = 133
- 调整后 = (4,200 × 0.095) / (2+3) = 79.8

更合理地反映竞争度
```

## 七、常见问题 FAQ

### Q1: 为什么不直接用"需供比"（Demand/Supply Ratio）？

**A**:
需供比 = 月搜索量 / 商品数

问题：
- "商品数"是搜索结果数，不等于真实竞争对手数
- 很多结果是相关性弱的产品
- 标题密度更精准（只看第一页优化情况）

---

### Q2: TES 和 SPR 有什么区别？

**A**:
- **TES**: 流量效能（收益/成本比），用于选词
- **SPR**: 排名预估单量（8天需要多少单才能上首页），用于销量目标

两者结合使用：
1. 用 TES 选出赢家词
2. 用 SPR 估算需要的销量
3. 制定广告预算和推广计划

---

### Q3: 我的产品是全新品类，所有词的标题密度都是 0，怎么办？

**A**:
恭喜你发现了蓝海市场！

策略：
- 此时所有词 TES 都会很高
- 应该按 "月搜索量 × 购买率" 的绝对值排序
- 优先选择搜索量 > 1000 且购买率 > 5% 的词

---

### Q4: TES 多久更新一次？

**A**:
建议频率：
- **月度复盘**: 每月 1 号下载最新数据，重新计算 TES
- **季度深度分析**: 每季度对比 TES 变化趋势
- **实时监控**: Top 10 核心词每周检查自然排名和标题密度

---

### Q5: TES 是否适用于所有品类？

**A**:
适用：
- ✅ 标品（如家居用品、电子配件）
- ✅ 搜索驱动型品类（用户主动搜索）

不适用：
- ❌ 冲动购买品类（如糖果、杂志）
- ❌ 推荐驱动型品类（用户主要通过 "Customers who bought" 购买）
- ⚠️ 时尚品类（搜索词变化快，TES 滞后）

## 八、总结

### TES 模型的本质

**TES 是一个"投资回报率"思维模型**:

```
TES = 收益 / 成本

收益 = 流量规模 (月搜索量) × 转化质量 (购买率)
成本 = 竞争激烈度 (标题密度)
```

**核心理念**:
> "不要追逐大流量，要追逐高效能流量"

### 实战应用三步法

**Step 1: 计算**
- 下载卖家精灵数据
- 套用公式批量计算 TES

**Step 2: 分层**
- TES > 100 → Winner 词（核心推广）
- TES 50-100 → Potential 词（次核心）
- TES 10-50 → Longtail 词（捡漏）
- TES < 10 → Weak 词（否定）

**Step 3: 执行**
- Listing 优化：赢家词优先埋入
- PPC 架构：按 TES 分层建立广告活动
- 持续监控：每月复盘，动态调整
