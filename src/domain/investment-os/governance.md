# Investment Reasoning OS Governance

## Scope

This domain is a **validation instance** for T1 cross-domain transferability.
It is NOT intended for production use or real financial decision-making.

## Forbidden Outputs

The following output types are strictly prohibited:

1. **Buy/Sell Recommendations** - No directional trade signals
2. **Price Predictions** - No target prices or ranges
3. **Timing Signals** - No "now is the time to..." statements
4. **Alpha Claims** - No promises of outperformance
5. **Specific Ticker Advice** - No "buy AAPL" type recommendations

## Allowed Outputs

1. **Reasoning Structures** - Causal chains, mechanism analysis
2. **Risk Frameworks** - Exposure identification, failure modes
3. **Trade-off Analysis** - Risk vs. return considerations
4. **Boundary Conditions** - When mechanisms apply or break down
5. **Falsifiability Criteria** - How to validate or invalidate hypotheses

## Compliance Statement

All experiments in this domain must comply with:

- No actionable trading outputs
- No financial advice implications
- Clear disclaimers on all generated content
- Focus on reasoning mechanism validation only
