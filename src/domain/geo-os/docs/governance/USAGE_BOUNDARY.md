> **Status**: CANONICAL

## 声明

本文档定义了 Geo-OS T1 Reasoning System（以下简称"系统"）的使用边界。
任何接触系统的人员必须理解并遵守这些边界。

---

## 第一条：系统性质声明

### 1.1 系统是什么

```
✅ 系统是：分析辅助工具
✅ 系统是：机制库存储和检索系统
✅ 系统是：结构化推理框架
✅ 系统是：可验证的知识组织方式
```

### 1.2 系统不是什么

```
❌ 系统不是：自动化决策系统
❌ 系统不是：预测工具
❌ 系统不是：效果保证服务
❌ 系统不是：商业咨询产品
❌ 系统不是：人工智能自主体
```

---

## 第二条：使用限制

### 2.1 禁止用途

使用者不得将系统用于以下用途：

| 禁止用途 | 说明 |
|----------|------|
| ❌ 自动执行商业决策 | 系统输出不得直接触发任何自动化操作 |
| ❌ 生产环境自动投放 | 不得将输出直接用于广告/定价等自动投放 |
| ❌ 作为效果承诺依据 | 不得引用系统输出作为效果保证 |
| ❌ 商业销售话术 | 不得在销售中声称"系统保证" |
| ❌ 替代专业判断 | 不得完全依赖系统替代专业人员判断 |

### 2.2 允许用途

```
✅ 辅助分析和理解
✅ 结构化思考框架
✅ 机制学习和参考
✅ 研究和教育目的
✅ 内部决策参考（非自动执行）
```

## 第三条：免责声明

### 3.1 不构成商业承诺

```
┌─────────────────────────────────────────────────────────┐
│                    重要声明                              │
│                                                         │
│  系统的任何输出均不构成：                                │
│  - 商业承诺                                              │
│  - 效果保证                                              │
│  - 投资建议                                              │

│                                                         │
│  使用者应自行承担使用系统输出的全部风险和责任。          │
└─────────────────────────────────────────────────────────┘
```

### 3.2 不保证效果

系统明确声明：

1. **不保证准确性**：机制描述基于有限样本，可能不适用于所有情况
2. **不保证完整性**：系统不覆盖所有可能的因素和变量
3. **不保证时效性**：平台规则和市场条件随时变化
4. **不保证普适性**：特定 Case 的结果不能推广到所有场景

### 3.3 不等同于自动化决策系统

```
系统输出需要人类：
- 验证适用性
- 结合具体情况
- 承担最终决策责任
- 监控执行效果

系统不具备：
- 自主决策能力
- 自动执行能力
- 效果预测能力
- 风险评估能力
```

## 第五条：输出使用规范

### 5.1 引用规范

如需引用系统输出，必须：

```
1. 注明来源："Geo-OS T1 Reasoning System 分析参考"
2. 添加免责："仅供参考，不构成任何承诺"
3. 说明局限："基于特定条件，可能不适用于其他情况"
```

### 5.2 禁止的引用方式

```
❌ "系统证明..."
❌ "根据 AI 分析，一定会..."
❌ "系统保证..."
❌ "自动化系统推荐..."
```

### 5.3 允许的引用方式

```
✅ "参考系统的分析框架..."
✅ "系统提供的机制参考显示..."
✅ "作为分析辅助，系统输出建议考虑..."
✅ "基于系统的结构化分析（仅供参考）..."
```

## 附录：常见问题

### Q: 我可以用系统输出来做商业决策吗？

A: 可以作为参考，但不能作为唯一依据，且必须经过人工验证和判断。系统不对决策后果负责。

### Q: 我可以告诉客户"这是 AI 推荐的"吗？

A: 不可以。这可能被误解为效果承诺。应说明"这是分析辅助工具的参考输出"。

### Q: 系统能自动优化我的广告吗？

A: 不能。系统不具备自动执行能力，所有操作必须由人工完成。
