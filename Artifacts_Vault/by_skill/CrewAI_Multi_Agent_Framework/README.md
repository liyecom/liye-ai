# CrewAI Multi-Agent Framework - Artifacts Vault

This directory stores execution artifacts from CrewAI crews for tracking, learning, and evolution.

## Directory Structure

```
CrewAI_Multi_Agent_Framework/
├── medical/          # Medical research crew outputs
├── ecommerce/        # Amazon/e-commerce optimization outputs
├── content/          # SEO content creation outputs
├── business/         # Business analysis outputs
├── multi-domain/     # Cross-domain or Flow-based outputs
└── README.md        # This file
```

## Artifact Naming Convention

**Format**: `YYYYMMDD_domain_topic_version.md`

**Examples**:
- `20251219_medical_HER2_treatment_v1.md`
- `20251220_ecommerce_laptop_stand_keywords_v1.md`
- `20251221_content_standing_desks_article_v2.md`

## Integration with Evolution Log

After saving an artifact:
1. Review quality and performance
2. Extract 2-3 key insights
3. Add entry to `evolution_log.md` in skill directory
4. Note any SOP improvements or template refinements needed

## Cross-Referencing

- **PARA System**: Reference in `20 Areas/` indexes if noteworthy
