# Hello World Example

A minimal example demonstrating how to use LiYe AI skills.

## Files

- `run.ts` - Simple skill execution example

## Running

```bash
# From project root
npx ts-node examples/hello-world/run.ts

# Or with Node.js
node examples/hello-world/run.js
```

## What it demonstrates

1. Loading a skill from the registry
2. Executing a skill with input
3. Handling skill output
