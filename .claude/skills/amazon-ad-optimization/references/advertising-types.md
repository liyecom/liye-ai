# Amazon广告类型参考指南

## 三种广告类型对比

| 维度 | SP (Sponsored Products) | SB (Sponsored Brands) | SD (Sponsored Display) |
|------|------------------------|----------------------|----------------------|
| **中文名** | 商品推广 | 品牌推广 | 展示型推广 |
| **广告位置** | 搜索结果中/产品详情页 | 搜索结果顶部 | 产品详情页/站外 |
| **广告格式** | 单个产品图 | 品牌logo+标题+3个产品<br>或视频 | 单个产品图/视频 |
| **品牌要求** | ❌ 不需要 | ✅ **必须有品牌备案** | ❌ 不需要 |
| **落地页** | 单个产品详情页 | 店铺/产品列表 | 产品详情页 |
| **主要目标** | **销量** | **品牌认知+销量** | **再营销/抢竞品** |
| **CTR基准** | 0.5-1.5% | **2-4%**（视频更高） | 0.3-0.8% |
| **成本** | 中 | **高**（但ROI好） | 低-中 |
| **适合阶段** | 所有阶段 | 品牌成熟期 | 有一定销量后 |
| **关键词投放** | ✅ 支持 | ✅ 支持 | ⚠️ 部分支持 |
| **自动投放** | ✅ 支持 | ❌ 不支持 | ✅ 支持 |

## SBV (Sponsored Brand Video) 说明

**核心认知**：
- **SBV = Sponsored Brands + Video格式**
- SBV不是独立的广告类型
- SBV是品牌推广（Sponsored Brands）的一种展现形式

**创建路径**：
```
Create campaign → Sponsored Brands → Video格式 ← 这就是SBV
```

**SBV vs SD Video对比**：

| 维度 | SBV（品牌推广视频） | SD视频 |
|------|-------------------|--------|
| 品牌要求 | ✅ 需要 | ❌ 不需要 |
| 广告位置 | 搜索结果顶部（Top of Search） | 产品详情页侧边栏 |
| CTR | **2-5%** | 0.5-1.0% |
| 成本 | 高 | 中低 |
| 效果 | **最好** | 一般 |

**结论**：如果有品牌备案，优先用SBV；如果没有，只能用SD视频。

## 广告组合策略

### 初期（新品上架）
- **只用SP**：70%自动投放（探索关键词）+ 30%手动投放（已知核心词）
- 预算：$50-100/天
- 目的：快速获取销量和评价

### 成长期（有品牌备案+一定销量）
- **SP（70%预算）**：继续获取销量，覆盖长尾关键词
- **SB（30%预算）**：抢Top of Search位置，建立品牌认知
- 如果有视频，优先用SBV

### 成熟期（稳定销量+品牌认知）
- **SP（50%预算）**：维持主力款销量
- **SB（30%预算）**：强化品牌，推新品
- **SD（20%预算）**：再营销流失客户，抢竞品流量

### 功能性产品+有视频（特殊情况）
- **SBV：60%预算** ← 因为CTR高
- SP：30%预算
- SD：10%预算

**适用产品**：需要展示功能的产品（如门垫吸水、防滑等）
