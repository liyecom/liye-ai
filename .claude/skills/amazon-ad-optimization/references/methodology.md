# Amazon广告优化核心方法论

## TES公式（Traffic Efficiency Score）

### 计算公式
```
TES = (月搜索量 × 购买率) / (标题密度 + 1)
```

### 参数说明
- **月搜索量**：关键词在Amazon的月度搜索次数
- **购买率**：搜索该关键词后实际产生购买的比例（%）
- **标题密度**：该关键词在同类产品标题中的出现频率（%）

### TES分级标准
- **🏆 WINNER** (TES > 100)：顶级关键词，必须优先布局
- **💎 POTENTIAL** (10 ≤ TES ≤ 100)：潜力关键词，值得投放
- **📊 BROAD** (TES < 10)：泛词，谨慎使用

### 应用场景
1. **标题优化**：优先放入TES>100的WINNER词
2. **广告投放**：WINNER词提高竞价20-30%
3. **流量缺口识别**：高TES但低标题密度的词（优化机会）

## 固定骨架+一子一主词策略

### 三大原则

**1. 固定骨架**
- 所有子ASIN核心结构必须一致
- 包含品类词、基础功能词、使用场景词
- 示例：`TIMO Door Mat Indoor Entrance Non Slip Waterproof`

**2. 一子一主词**
- 每个子ASIN只多吃1个差异化属性词
- 差异化词必须是"属性"而非"用途"
- 示例：Absorbent、Low Profile、Machine Washable

**3. CEO判断法**
- 测试问题："如果把这个标题复制给所有子ASIN，逻辑是否全部成立？"
- ✅ 成立 → 合规
- ❌ 不成立 → 违规（可能被判变体滥用）

### 关键词分类体系

| 类型 | 是否可用于差异化 | 示例 |
|------|----------------|------|
| **品类词** | ❌ 必须统一 | door mat, rug |
| **功能属性词** | ✅ 可拆分 | absorbent, washable, low profile |
| **使用场景词** | ✅ 可拆分 | indoor, outdoor, entryway |
| **人群/用途词** | ❌ 禁止（风险高） | for dogs, for kids, decorative |

### 合规性验证（CEO判断法）

**测试1：属性互换测试**
- 问题："如果我把ASIN A的差异化词换成ASIN B的，逻辑是否成立？"
- ✅ 成立 → 继续
- ❌ 不成立 → 修改

**测试2：品类一致性测试**
- 问题："所有子ASIN的核心品类词是否一致？"
- ✅ 一致 → 继续
- ❌ 不一致 → 修改

**测试3：用途一致性测试**
- 问题："是否有子ASIN暗示了不同的使用人群或用途？"
- ✅ 没有 → 通过
- ❌ 有 → 修改

## 3-3-3广告结构方案

### 结构说明
- **3个广告活动**
- **每个活动3个广告组**
- **每个广告组3个核心关键词**

### 标准配置

**活动1：大词攻坚**（预算占比60%）
- 目标：抢占高搜索量核心词
- 关键词：TES>1000的S级关键词
- 竞价策略：高竞价（比建议竞价+30-50%）
- ASIN选择：主力款（CVR最高的）

**活动2：精准场景**（预算占比25%）
- 目标：覆盖中等搜索量场景词
- 关键词：TES 300-1000的A级关键词
- 竞价策略：中竞价（建议竞价±10%）
- ASIN选择：主力款+副款

**活动3：长尾探索**（预算占比15%）
- 目标：测试长尾关键词
- 类型：Automatic Targeting
- 竞价策略：低竞价（建议竞价-20%）
- ASIN选择：所有ASIN

### 预算分配示例（总预算$150）

```
活动1：大词攻坚 - $90
  ├─ 广告组1：品类核心词（$40）
  ├─ 广告组2：功能场景词（$30）
  └─ 广告组3：竞品词（$20）

活动2：精准场景 - $40
  ├─ 广告组1：使用场景A（$15）
  ├─ 广告组2：使用场景B（$15）
  └─ 广告组3：功能属性词（$10）

活动3：长尾探索 - $20
  └─ 自动投放（$20）
```

## CTR优化基准和策略

### CTR基准数据

| 广告类型 | 行业基准 | 优秀水平 | 警戒线 |
|---------|---------|---------|-------|
| SP（图片） | 0.5-1.5% | >1.5% | <0.5% |
| SBV（视频） | 2-4% | >4% | <2% |
| SD（图片） | 0.3-0.8% | >0.8% | <0.3% |
| SD（视频） | 0.5-1.0% | >1.0% | <0.5% |

### CTR优化紧急方案（CTR<0.5%时）

**立即行动**（当天执行）：
1. **提高竞价40-50%** → 提升广告位
2. **创建SBV活动**（如有视频素材）→ CTR提升5倍
3. **优化主图** → 使用场景图、对比图
4. **调整预算分配** → SBV占60%，SP占30%

**预期效果**：
- SP CTR：从0.47%提升到0.8-1.2%
- SBV CTR：2-3%
- 整体CTR提升3-5倍

### 视频广告优势

**数据验证**：
- 视频广告CTR：2.20%
- 图片广告CTR：0.47%
- **差距：5倍**

**适用产品**：
- 功能性产品（需要演示）
- 视觉差异化小的产品
- 使用场景需要说明的产品

**视频规范**：
- 时长：6-45秒（推荐15-30秒）
- 比例：16:9（横屏）或1:1（方形）
- 分辨率：≥1920x1080
- 格式：MP4或MOV
- 大小：<500MB

## 每日数据跟踪指标

### 核心指标（每日必看）
1. **曝光（Impressions）**：广告展示次数
2. **点击（Clicks）**：用户点击次数
3. **CTR（Click-Through Rate）**：点击率 = 点击/曝光
4. **花费（Spend）**：广告支出
5. **CPC（Cost Per Click）**：每次点击成本 = 花费/点击
6. **订单（Orders）**：广告带来的订单数
7. **销售额（Sales）**：广告带来的销售金额
8. **CVR（Conversion Rate）**：转化率 = 订单/点击
9. **ACOS（Advertising Cost of Sales）**：广告成本占比 = 花费/销售额

### 关键节点复盘
- **Day 3**：首次验证数据，判断是否需要调整
- **Day 7**：首周复盘，评估整体策略
- **Day 14**：双周复盘，决定是否扩展/收缩
- **Day 30**：最终复盘，总结经验教训

### 异常处理规则

**CTR异常低（<0.5%）**：
- 提高竞价30-50%
- 检查主图质量
- 查看Search Term Report，加否定词

**ACOS过高（>40%）**：
- 降低竞价10-20%
- 暂停高ACOS关键词
- 检查产品页转化率

**CVR过低（<3%）**：
- 优化产品页（标题、主图、bullet points）
- 检查价格竞争力
- 增加评价数量

**花费过快（预算提前耗尽）**：
- 降低竞价
- 减少关键词数量
- 添加更多否定词
