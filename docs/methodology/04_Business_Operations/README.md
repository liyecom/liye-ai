# 04 Business Operations

Business process and operations management skills.

## Domain

Skills in this domain enable agents to:
- Manage business workflows
- Optimize operational processes
- Track KPIs and metrics
- Coordinate cross-functional activities

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `process-optimization` | Workflow analysis and improvement | Planned |
| `kpi-tracking` | Metrics monitoring and reporting | Planned |
| `resource-allocation` | Resource planning and assignment | Planned |
| `project-coordination` | Cross-team project management | Planned |

## Usage

```yaml
# In agent definition
skills:
  atomic:
    - business_operations/process_optimization
    - business_operations/kpi_tracking
```
