## 🔹01. Skill Identity（技能身份）

**Skill Name**: CSV Data Summarizer / CSV 数据自动洞察

**Core Mission**:
自动分析 CSV 文件并生成综合洞察报告，无需用户手动提示即可识别数据模式、异常值、趋势，并创建可视化，是通用数据分析的快速启动工具。

**Capability Domain**: Data Science
- 自动数据摘要生成
- 统计分析（描述性/推断性）
- 异常值与模式检测
- 趋势识别与预测
- 自动可视化生成

**Target Scenarios**:
- 快速理解未知数据集
- 数据质量评估
- 初步探索性数据分析 (EDA)
- 报表数据洞察
- 数据驱动决策支持

## 🔹04. Methods & SOPs（方法论 / 操作手册）

### Standard Operating Procedure: Auto CSV Summarization

#### Phase 1: 数据加载与理解
```
Step 1.1 加载数据
  - 读取 CSV 文件
  - 处理编码问题
  - 识别分隔符

Step 1.2 数据概览
  - 行数、列数
  - 列名列表
  - 数据类型推断
  - 内存占用

Step 1.3 数据质量评估
  - 缺失值统计
  - 唯一值统计
  - 数据类型一致性
```

#### Phase 2: 自动统计分析
```
Step 2.1 数值列分析
  - 均值、中位数、众数
  - 标准差、方差
  - 最小值、最大值
  - 四分位数
  - 偏度、峰度

Step 2.2 分类列分析
  - 唯一值数量
  - 频率分布
  - 众数
  - 缺失率

Step 2.3 时间列分析
  - 时间范围
  - 时间间隔
  - 周期性检测
```

#### Phase 3: 模式与异常发现
```
Step 3.1 相关性分析
  - 数值列相关矩阵
  - 高相关性对识别
  - 多重共线性检测

Step 3.2 异常值检测
  - IQR 方法
  - Z-score 方法
  - 隔离森林（可选）

Step 3.3 趋势识别
  - 线性趋势
  - 季节性模式
  - 突变点检测
```

#### Phase 4: 可视化与报告
```
Step 4.1 自动图表生成
  - 每个数值列：直方图 + 箱线图
  - 每个分类列：柱状图
  - 相关性：热力图
  - 时序：折线图

Step 4.2 洞察报告生成
  - 数据概览摘要
  - 关键统计发现
  - 异常值清单
  - 模式与趋势
  - 建议与下一步
```

## 🔹05. Execution Protocols（执行协议）

### Pre-Execution Checklist

**自动执行，无需询问**：
- ✓ 数据加载
- ✓ 基础统计
- ✓ 异常检测
- ✓ 可视化生成

**可选确认**：
- ? 深度分析需求（相关性、聚类等）
- ? 特定列聚焦
- ? 输出格式偏好

### Decision-Making Logic

**小数据集 (<10,000 行)**：
→ 全量分析
→ 完整可视化

**中数据集 (10K-1M 行)**：
→ 采样分析 + 全量统计
→ 关键可视化

**大数据集 (>1M 行)**：
→ 分层采样
→ 聚合统计
→ 提示用户选择聚焦区域
