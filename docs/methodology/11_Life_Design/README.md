# 11 Life Design

Personal life planning and design skills.

## Domain

Skills in this domain enable agents to:
- Set and track personal goals
- Design life systems and routines
- Balance multiple life domains
- Plan long-term life trajectory

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `goal-setting` | SMART goal creation and tracking | Planned |
| `life-audit` | Life domain assessment and balance | Planned |
| `system-design` | Personal productivity system creation | Planned |
| `vision-planning` | Long-term vision and milestone planning | Planned |

## Usage

```yaml
# In agent definition
skills:
  atomic:
    - life_design/goal_setting
    - life_design/system_design
```
