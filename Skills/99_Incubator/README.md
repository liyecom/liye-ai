# 99_Incubator

> **状态**: `ACTIVE` - 实验性技能孵化器

## 定位

孵化器域 - 用于实验性技能的开发与测试，成熟后迁移到正式域。

## 使用规则

1. 所有实验性技能先在此域开发
2. 通过验证后迁移到对应的正式域
3. 保留失败实验的文档以供学习

## 当前实验

- `test-skill/` - 技能开发测试模板

## 毕业条件

实验技能满足以下条件可毕业到正式域：
1. 完成功能验证
2. 编写完整文档
3. 通过代码审查
4. 确定目标域归属
