# 02 Analysis Strategy

Strategic analysis and decision-making skills.

## Domain

Skills in this domain enable agents to:
- Perform competitive analysis
- Develop strategic frameworks
- Evaluate options and trade-offs
- Generate actionable recommendations

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `competitive-analysis` | Analyze competitors and market positioning | Planned |
| `swot-analysis` | Strengths, Weaknesses, Opportunities, Threats | Planned |
| `decision-matrix` | Multi-criteria decision analysis | Planned |
| `scenario-planning` | Future scenario development and analysis | Planned |

## Usage

```yaml
# In agent definition
skills:
  atomic:
    - analysis_strategy/competitive_analysis
    - analysis_strategy/decision_matrix
```
