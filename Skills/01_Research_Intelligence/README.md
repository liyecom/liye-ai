# 01 Research Intelligence

Research and information gathering skills for systematic knowledge acquisition.

## Domain

Skills in this domain enable agents to:
- Conduct systematic literature reviews
- Gather intelligence from multiple sources
- Validate and cross-reference information
- Synthesize findings into actionable knowledge

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `web-research` | Systematic web search and information extraction | Planned |
| `academic-search` | Academic database queries (PubMed, Scholar, etc.) | Planned |
| `source-verification` | Credibility assessment and fact-checking | Planned |
| `knowledge-synthesis` | Combining multiple sources into coherent summaries | Planned |

## Usage

```yaml
# In agent definition
skills:
  atomic:
    - research_intelligence/web_research
    - research_intelligence/source_verification
```
