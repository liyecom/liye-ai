# 10 Health Wellness

Personal health and wellness management skills.

## Domain

Skills in this domain enable agents to:
- Track health metrics
- Design wellness routines
- Provide lifestyle recommendations
- Monitor habit formation

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `health-tracking` | Metrics monitoring and analysis | Planned |
| `routine-design` | Daily routine optimization | Planned |
| `habit-coaching` | Behavior change support | Planned |
| `wellness-planning` | Holistic wellness strategy | Planned |

## Usage

```yaml
# In agent definition
skills:
  atomic:
    - health_wellness/health_tracking
    - health_wellness/routine_design
```
