# 07 Data Science

Data analysis, modeling, and machine learning skills.

## Domain

Skills in this domain enable agents to:
- Perform statistical analysis
- Build and evaluate models
- Process and transform data
- Generate data-driven insights

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `statistical-analysis` | Descriptive and inferential statistics | Planned |
| `data-visualization` | Chart and dashboard creation | Planned |
| `predictive-modeling` | ML model development and evaluation | Planned |
| `data-pipeline` | ETL and data transformation | Planned |

## Usage

```yaml
# In agent definition
skills:
  atomic:
    - data_science/statistical_analysis
    - data_science/data_visualization
```
