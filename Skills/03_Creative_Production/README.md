# 03 Creative Production

Content creation and creative output skills.

## Domain

Skills in this domain enable agents to:
- Generate written content (articles, copy, documentation)
- Create visual concepts and designs
- Develop creative campaigns
- Produce multimedia content

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `content-writing` | Article, blog, and documentation writing | Planned |
| `copywriting` | Marketing and advertising copy | Planned |
| `visual-concept` | Design briefs and visual direction | Planned |
| `campaign-design` | Marketing campaign ideation | Planned |

## Usage

```yaml
# In agent definition
skills:
  atomic:
    - creative_production/content_writing
    - creative_production/copywriting
```
