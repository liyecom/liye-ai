# 08 Communication

Communication, presentation, and messaging skills.

## Domain

Skills in this domain enable agents to:
- Craft clear and effective messages
- Create presentations and reports
- Facilitate discussions and meetings
- Manage stakeholder communication

## Skills

| Skill | Description | Status |
|-------|-------------|--------|
| `message-crafting` | Clear, audience-appropriate messaging | Planned |
| `presentation-design` | Slide deck and visual presentation | Planned |
| `report-generation` | Structured report writing | Planned |
| `meeting-facilitation` | Agenda creation and summary | Planned |

## Usage

```yaml
# In agent definition
skills:
  atomic:
    - communication/message_crafting
    - communication/report_generation
```
