#### **设置匹配类型**

**页面: 关键词列表**

对每个关键词设置匹配类型：

| 关键词 | 匹配类型 | 理由 |
|--------|----------|------|
| **tapetes para entrada de casa** | Exact Match（精准匹配） | 🔥 TES 133，转化率高，只投精准 |
| **tapetes para puerta** | Exact Match（精准匹配） | 🔥 TES 254，最高效词，精准投放 |
| **alfombra de entrada** | Phrase Match（词组匹配） | ⚠️ TES 91，用词组捕捉变体（如 "alfombra de entrada moderna"） |

**匹配类型选择逻辑**:
```
Exact Match（精准）:
- 用于高 TES 词（>100）
- 只在用户搜索完全匹配时展示
- CPC 低，转化率高

Phrase Match（词组）:
- 用于中 TES 词（50-100）
- 允许前后有其他词（如 "alfombra de entrada grande"）
- 捕捉长尾变体

Broad Match（广泛）:
- ❌ 测试期不推荐
- 会匹配大量不相关搜索
```

## 🔧 常见问题与解决方案

### Q1: 投放后 6 小时，展示量 = 0

**可能原因**:
1. ❌ 关键词拼写错误（西班牙语特殊字符问题）
2. ❌ 竞价过低（低于最低展示阈值）
3. ❌ ASIN 状态异常（产品被下架或限制）

**解决方案**:
```
步骤 1: 检查关键词拼写
- tapetes（正确）vs tapate（错误）
- entrada（正确）vs enterada（错误）

步骤 2: 竞价翻倍测试
- $0.50 → $1.00
- 观察 2 小时

步骤 3: 检查 ASIN 状态
- 赛狐后台 → 产品管理 → 查看 B08SWLTTSW 状态
- 如果显示 "Inactive" → 联系 Amazon 客服
```

---

### Q2: CTR 很低（< 1.5%）

**可能原因**:
主图吸引力不足，西班牙语用户有不同审美偏好

**解决方案**:
```
立即优化主图：
1. 添加西班牙语文字（如 "Lavable" "Antideslizante"）
2. 使用暖色调（西班牙语用户偏好）
3. 展示家庭使用场景（不是单品展示）

测试方法：
- A/B Test: 上传新主图，观察 CTR 变化
- 如果 CTR 提升 > 1%，说明有效
```

---

### Q3: 有点击但 0 订单（CVR = 0%）

**可能原因**:
1. 价格过高（西班牙语用户价格敏感）
2. Listing 无西班牙语内容（语言不匹配）
3. 评论太少（信任度不足）

**解决方案**:
```
优先级 1: 检查价格
- 对比同类产品价格
- 如果高于平均 15% → 降价或设置优惠券

优先级 2: 添加西班牙语 A+ 内容
- 至少添加 1 个西班牙语模块
- 翻译 5 个核心卖点

优先级 3: 增加评论
- 使用 Amazon Vine 计划
- 或运行 "订单后索评" 自动化
```

### Q4: 某个词 ACOS 极低（< 15%），其他词 ACOS 极高
